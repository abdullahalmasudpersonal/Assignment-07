JavaScript is a client side scripting or browser scripting language. It is designed with the goal of controlling the behavior at the client level of the website. But now it runs on any operating system except browsers and it has all the features of other high level programming languages. JavaScript is called JS for short. JavaScript is a prototype-based scripting language that has variable, weak type and first class functions. JavaScript is a language that supports an object oriented, functional programming style. JavaScript was launched on Netscape, a web browser built in the 1990s.

JavaScript is one of the most commonly used scripting languages. The term client-side scripting language means it runs client-side (or on the client machine) inside the web-browser, but one important thing to keep in mind is that the client's web-browser needs to support JavaScript or not. JavaScript must be enabled.Nowadays, most modern web browsers support JavaScript and have JavaScript engine. For example, Google Chrome has its own JavaScript engine called V8.

javascript is an interpreted language. Not a compiled language. Compiled language is harder than interpreted language. The source code of the compiled language is passed through a program called a compiler, which translates it into bytecodes. Which the machine can understand and operate. The way JavaScript works is interesting. Inside a normal Web page you place some JavaScript code (See How Web Pages Work for details on Web pages). When the browser loads the page, the browser has a built-in interpreter that reads the JavaScript code it finds in the page and runs it.
